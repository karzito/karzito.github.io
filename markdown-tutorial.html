[Markdown Tutorial] 마크다운 기본 문법 정리

Github blog posting 하는데 필요한 markdown 기본 문법에 대해서 정리할 예정이다. 일단은 당장 필요한 간단한 문법 정도만 정리하고, 추가적으로 필요한 문법들 알아갈때마다 추가적으로 정리하도록 한다.

아래의 글은 마크다운 관련 포스팅을 참고하여 필요한 부분을 정리하였으며, 참조링크는 문서 하단에 표기하였다.

1. 헤더 Headers

  • Title

    This is a title
====================

This is a title

  • Subtitle

    This is a subtitle
------------------

This is a subtitle

  • Bullet: 1~6개 까지만 지원

    # This is a bullet1
## This is a bullet2
### This is a bullet3
#### This is a bullet4
##### This is a bullet5
###### This is a bullet6

This is a bullet1

This is a bullet2

This is a bullet3

This is a bullet4

This is a bullet5
This is a bullet6

2. Blockquote

  > This is a first blockquote.
  >> This is a second blockquote.
  >>> This is a third blockquote.

This is a first blockquote.

This is a second blockquote.

This is a third blockquote.

3. List

  • Ordered list

    1. first
2. second
3. third
  1. first
  2. second
  3. third

  • Unordered list

    * first
  * second
    * third

+ first
  + second
    + third

- first
  - second
    - third
  • first
    • second
      • third
  • first
    • second
      • third
  • first
    • second
      • third

4. Code

4.1 들여쓰기

3개의 tab을 이용해서 들여쓰기를 만나면 변환되기 시작해서 들여쓰지 않은 행을 만날떄까지 변환이 계속된다.

  Normal paragraph.

      First code line.
      Second code line.

  End code block.

Normal paragraph.

  First code line.
  Second code line.

End code block.

4.2 Code 강조 숫자 1번 키 왼쪽에 있는 ` (Grave)를 입력하세요.

  This is `Grave`
  ``This is `Grave` ``

This is Grave This is `Grave`

4.3 Code Block

Grave와 language명을 이용하여 다음과 같이 코드를 작성한다.

  ```c
   typedef struct {
     int a;
     int b;
     int c;
   } Alphabet_t
  ```

typedef struct {
  int a;
  int b;
  int c;
} Alphabet_t

5. Horizontal Line

아래의 줄은 모두 수평선을 만든다.

  * * *
  ***
  *****
  - - -
  ----------------------

6. 강조

  __double underscores__
  ~~cancelline~~

double underscores cancelline

7. Link

  [google](https://www.google.com/)
  
  [naver](https://www.naver.com/ "마우스 대면 설명 나옴")

  () 대신 []를 이용하여 링크를 저장할 수도 있다.

  [GOOGLE][GOOGLE_LINK]

  [GOOGLE_LINK]: https://www.google.com/

google

naver
